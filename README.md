# 🚀 Clone da Landing Page Zoom

[🔗 Acessar projeto online](https://adamasnegro.github.io/Zoom-Estrutura-com-Grid/)

Projeto desenvolvido como prática dos estudos em **Análise e Desenvolvimento de Sistemas (ADS)**, com foco em **HTML5**, **CSS3**, **CSS Grid**, **Flexbox** e **Responsividade**.

## 📋 Sobre o Projeto

Este projeto consiste na criação de uma landing page inspirada no site da Zoom, com o objetivo de praticar conceitos fundamentais de desenvolvimento front-end.

Durante o desenvolvimento foram aplicados conceitos como:

* Estruturação semântica com HTML5
* Layout utilizando CSS Grid
* Alinhamento com Flexbox
* Design responsivo com Media Queries
* Organização e reutilização de estilos CSS
* Boas práticas de estrutura de código

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Media Queries

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em diferentes tamanhos de tela, adaptando o layout para dispositivos móveis, tablets e desktops.

## 🚀 Como Executar

1. Clone este repositório:

```bash
git clone git@github.com:adamasnegro/Zoom-Estrutura-com-Grid.git
```

2. Acesse a pasta do projeto:

```bash
cd nome-do-repositorio
```

3. Abra o arquivo `index.html` no navegador.
